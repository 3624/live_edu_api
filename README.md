## 直播云课堂

使用了百家云的直播api构建的一个简单的教育直播网站。地址：http://live.bobcheng.space

### 技术栈

#### 前端

- html/css/JavaScript
- jQuery
- [art-template](https://github.com/aui/art-template) （前端js渲染引擎）

#### 后端

- php/mysql
- [thinkphp5](http://www.thinkphp.cn/) (php框架)

### 功能

#### 教师

- 创建普通大班直播房间
- 发放邀请码
- 上传课件
- 进行授课
- 录制直播回放
- 上传教学视频

#### 学生

- 首页进入直播
- 邀请码进入直播
- 观看回放（需要老师录制）
- 观看点播（老师上传的视频）
- 观看历史记录
